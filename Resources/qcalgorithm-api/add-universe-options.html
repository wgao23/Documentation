    Symbol                                                  underlyingSymbol,
    Func<OptionFilterUniverse, OptionFilterUniverse>  optionFilter

Adds a new universe that creates options of the security by monitoring any changes in the Universe the provided security is in. Additionally, a filter can be applied to the options generated when the universe of the security changes.

Parameters
Symbol underlyingSymbol Underlying Symbol to add as an option. For Futures, the option chain constructed will be per-contract, as long as a canonical Symbol is provided.
Func<OptionFilterUniverse, OptionFilterUniverse> optionFilter User-defined filter used to select the options we want out of the option chain provided.

Return

Void - This method provides no return.
